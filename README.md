# Vrinda-Retail-Store-Excel
This project focuses on analyzing retail store sales data using Excel dashboards and visualizations to generate meaningful business insights. The analysis helps to understand customer behavior, sales trends, and channel performance for the year 2022.

The project answers key business questions using an interactive Excel dashboard built with Pivot Tables, Charts, and Slicers.
